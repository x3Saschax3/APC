ACPCommands Modul

Das ACPCommands-Modul ermöglicht es berechtigten Nutzern (Channel-Besitzer und App-Manager), das Admin Control Panel (ACP) zu öffnen, um administrative Aufgaben zu erledigen.

Funktionen und Logik

Überprüfung und Erstellung von App.modules

Falls App.modules noch nicht existiert, wird es als leeres Objekt erstellt, um Fehler zu vermeiden, falls das ACPCommands-Modul als erstes Modul geladen wird.

enabled-Eigenschaft

Die Eigenschaft enabled ist standardmäßig auf true gesetzt. Wenn enabled auf false gesetzt ist, wird das Modul deaktiviert, und Benutzer können nicht auf das ACP zugreifen.

Funktion acp

Diese Funktion öffnet das Admin Control Panel für berechtigte Benutzer.

  • Berechtigungsprüfung: Nur der Channel-Besitzer und App-Manager dürfen das ACP öffnen.
  • Falls der Benutzer berechtigt ist, wird die HTML-Datei acp.html geladen und als Popup angezeigt.
  • Nachricht bei fehlender Berechtigung: Falls der Benutzer keine Berechtigung hat, wird ihm eine entsprechende private Nachricht angezeigt.

Zusammenfassung

Das ACPCommands-Modul ermöglicht es dem Channel-Besitzer und App-Manager, das Admin Control Panel zu öffnen, um administrative Aufgaben zu verwalten. Über die enabled-Eigenschaft kann das Modul aktiviert oder deaktiviert werden. Berechtigungen werden überprüft, und nur autorisierte Benutzer können auf das ACP zugreifen.
